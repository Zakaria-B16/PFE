PV Sizer

Welcome To PV Sizer

Proper sizing of an installation is an essential step to master, for this, we created a web app to make a pv system sizing just by inserting your location and desired charges and you, easy to use with precise results

Our Work

Our work consists in realizing an interface for the dimensioning of the autonomous photovoltaic installations in any place in the world, we used HTML as a rendering language and JavaScript to receive and manage the data and do the calculations. The customer must enter the location where he wants to install perform his installation, the number of days of autonomy and its charges with their powers, the time of daily use. Then we used an API (Application Programming Interface) to retrieve the latitude and longitude of the place entered with which we will have the daily global sunshine from another API also specialized in this field.
